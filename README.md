# MoviesProject 🎬

[cite_start]Movies Project is a web application created to help movie lovers manage and explore their favorite films[cite: 3]. [cite_start]It serves as a personal, digital movie collection where you can save films you've watched, rate them, and view their details in a modern and engaging interface[cite: 4].

## ✨ Core Features

* [cite_start]**Visual Movie Collection:** Displays your movies in a stylish card-based layout, with each movie featuring its own poster[cite: 6].
* [cite_start]**Poster Uploads:** Allows users to upload an image file for each movie poster, creating a personalized and rich visual library[cite: 7].
* [cite_start]**Full CRUD Functionality:** Provides all essential operations: Create, Read, Edit, and Delete movies from the collection[cite: 8].
* [cite_start]**Dark/Light Mode:** Includes a theme toggle in the navigation bar, allowing for a comfortable viewing experience in different lighting conditions[cite: 9].

## 🖥️ User Interface & Pages

* [cite_start]**Home Page:** The application welcomes users with a cinematic landing page featuring a full-screen background image[cite: 13]. [cite_start]It includes a clear headline, a short description of the project, and a prominent "Explore Movies" button that navigates to the main collection[cite: 14].
* [cite_start]**Movies Collection (Index Page):** Your movies are presented as a grid of visually appealing cards[cite: 22]. [cite_start]Each card displays the movie's poster, title, rating, and other key details, along with buttons to Edit, view Details, or Delete it[cite: 23].
* [cite_start]**Create New Movie Page:** This page provides a clean, user-friendly form to add a new movie to the collection[cite: 37]. [cite_start]You can upload an image file directly from your computer to be displayed with the movie's entry[cite: 38].
* [cite_start]**Edit Movie Page:** The Edit page allows you to update the details of a movie you've already added[cite: 51]. [cite_start]The form is pre-filled with the current information, making it easy to correct the genre, release date, or rating for any film in your library[cite: 52].
* [cite_start]**Movie Details Page:** This page provides a focused, read-only view of a single movie's information[cite: 65]. [cite_start]It presents the title, release date, genre, and rating in a clean format[cite: 66].
* [cite_start]**Delete Confirmation Page:** To prevent accidental deletions, this page acts as a final checkpoint[cite: 74]. [cite_start]It shows the details of the movie you're about to remove and requires a final confirmation click before the record is permanently removed[cite: 75].

## 🛠️ Tech Stack (Example)
* **Backend:** C#, .NET, Entity Framework Core
* **Database:** SQL Server
* **Frontend:** HTML, CSS, JavaScript, Bootstrap (or your chosen framework)
* **Architecture:** MVC / Clean Architecture

## 🚀 Getting Started

1. Clone the repository: `git clone https://github.com/your-username/MoviesProject.git`
2. Update the database connection string in `appsettings.json`.
3. Open the solution in Visual Studio or your preferred IDE.
4. Run the Entity Framework migrations to set up the database: `Update-Database`.
5. Run the application (F5).
